# Options

* `!sayHello ${Username}`

Create a wave for any member of your server.

### Example
    `!sayHello @useless bot`

## Get gay method

* `!howGay ${Username}`

Generates a *random* number for how gay is the mentioned user, **for fun**.

### Example
